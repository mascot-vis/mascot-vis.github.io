A Scene-Centric Approach

Instead of asking how data is transformed into a visualization,
Mascot asks how a graphical scene is constructed, modified, and made interactive with data as a key input.
Mascot provides an explicit model for semantic objects such as marks, collections, encodings, layouts, and axes, and automatically manages the dependencies among them. This lets changes to one object update the related parts of the scene while keeping you above low-level graphical primitives.

Construct expressive data graphics.

Mascot builds scenes from primitive marks through semantic operations such as repeat, divide, encode, layout, and align. Instead of producing an opaque chart, each operation creates objects that can be referenced and modified later.

Reactive scene modification

Mascot understands dependencies among semantic objects, so when you change an axis, encoding, layout, or collection, the related marks, guides, and labels automatically update.
